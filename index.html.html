📱 How to Install as an App

This lab is a PWA (Progressive Web App) — it can run like a native desktop or mobile app, completely offline.

🖥️ Desktop (Chrome / Edge)
1. Open the HTML file in Chrome or Edge
2. Look for the install icon ⊕ in the address bar
3. Click it → "Install" → Done!
4. Or: Menu (⋮) → "Save and share" → "Install page as app"
📱 iPhone / iPad (Safari)
1. Host the file (GitHub Pages, Netlify, etc.)
2. Open in Safari
3. Tap Share ⎋ → "Add to Home Screen"
4. It opens fullscreen, no browser bar!
🤖 Android (Chrome)
1. Host the file online
2. Open in Chrome → tap ⋮ menu
3. "Add to Home screen" → Install
4. Runs as standalone app!
🌐 Free Hosting Options
GitHub Pages — free, easy, fast
1. Create repo → upload HTML file
2. Settings → Pages → Deploy from branch
3. Your URL: username.github.io/repo-name

Netlify — drag and drop the HTML file at netlify.com/drop
Vercel — also free, very fast CDN
💾 Your Data
All data is saved in IndexedDB — your browser's local database.
✓ Survives page refreshes
✓ Works offline once loaded
✓ Tied to the domain/file you open it from
⚠ Clearing browser data = clearing your notes
